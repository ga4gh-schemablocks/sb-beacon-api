* type: string

Use `CUSTOM` for the `id` when no ontology is available.

##### `id` Value Example  

```
"EFO:0004157"
```

#### label

* type: string

This would be the "preferred Label" in the case of an ontology term.

##### `label` Value Example  

```
"BAM format"
```
